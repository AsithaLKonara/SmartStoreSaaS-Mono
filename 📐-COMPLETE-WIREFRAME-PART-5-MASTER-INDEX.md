## üîó BUTTON ‚Üí API ‚Üí PAGE MAPPING

### Primary Actions by Page Type

#### Product Pages
- **[+ Add Product]** ‚Üí POST `/api/products` ‚Üí `/dashboard/products/[id]`
- **[Publish]** ‚Üí POST `/api/products` (isActive=true) ‚Üí `/dashboard/products`
- **[Save Draft]** ‚Üí POST `/api/products` (isActive=false) ‚Üí Same page
- **[Delete]** ‚Üí DELETE `/api/products/[id]` ‚Üí `/dashboard/products`

#### Order Pages
- **[+ Create Order]** ‚Üí POST `/api/orders` ‚Üí `/dashboard/orders/[id]`
- **[Process Payment]** ‚Üí POST `/api/payments/confirm` ‚Üí Confirmation
- **[Cancel Order]** ‚Üí PUT `/api/orders/[id]` (status=CANCELLED) ‚Üí Same page
- **[Refund]** ‚Üí POST `/api/payments/refund` ‚Üí Same page

#### Customer Pages
- **[+ Add Customer]** ‚Üí POST `/api/customers` ‚Üí `/dashboard/customers/[id]`
- **[Edit]** ‚Üí PUT `/api/customers/[id]` ‚Üí Same page
- **[Delete]** ‚Üí DELETE `/api/customers/[id]` ‚Üí `/dashboard/customers`
- **[Send Email]** ‚Üí POST `/api/email/send` ‚Üí Modal close

#### Inventory Pages
- **[Adjust Stock]** ‚Üí POST `/api/inventory/[id]/adjust` ‚Üí Reloads
- **[Transfer]** ‚Üí POST `/api/inventory/transfer` ‚Üí Reloads
- **[Import]** ‚Üí POST `/api/import/inventory` ‚Üí Reloads
